# 🍽️ Culina - Cook What you Have

Culina is a full-stack recipe web application where users can explore delicious recipes with beautiful food images, search dishes, and enjoy a clean modern cooking experience.

## 🚀 Features

- Browse multiple recipes
- Beautiful recipe cards with food images
- Search functionality
- Responsive modern UI
- Dynamic image generation using Pexels API
- Full-stack architecture
- Clean and user-friendly design

## 🛠️ Tech Stack

### Frontend
- React.js
- CSS
- JavaScript
- Vite

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### APIs
- Pexels API

## 📂 Project Structure

Culina/
├── frontend/
├── backend/
└── README.md

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/mitalikhamkar/Culina.git
```

### Install frontend dependencies

```bash
cd frontend
npm install
```

### Install backend dependencies

```bash
cd ../backend
npm install
```

### Run backend

```bash
npm start
```

### Run frontend

```bash
cd ../frontend
npm run dev
```
